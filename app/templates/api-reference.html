Complete documentation for integrating with our chatbot API

Quick Start

Our API uses RESTful endpoints and returns JSON responses. All requests require an API key which you can find in your dashboard.

Base URL:
https://yourdomain.com
Authentication:

Include your API key in the request body for all API calls:

{
  "api_key": "your_bot_api_key_here"
}
Chat API Endpoint
POST

URL: /api/chat

Send messages to your chatbot and receive AI-generated responses.

Request Headers:
Content-Type: application/json
Request Body:
{ "api_key": "your_bot_api_key_here", "query": "User's message here" }
Success Response (200):
{ "response": "Bot's answer based on trained content", "error": null }
Error Responses:
// Invalid API Key (401) { "response": null, "error": "Invalid API key" } // Missing Parameters (400) { "response": null, "error": "API key and query are required" }
One-time Ask API
POST

URL: /api/ask

Ask questions about any website without creating a bot first (for testing purposes).

Request Body:
{ "url": "https://example.com", "query": "Your question about the website" }
HTTP Status Codes
Code Description Meaning
200 Success Request completed successfully
400 Bad Request Missing or invalid parameters
401 Unauthorized Invalid or missing API key
500 Internal Server Error Server-side issue

JavaScript Fetch Example:
// Send message to chatbot async function sendMessage(message) { const response = await fetch('/api/chat', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ api_key: 'YOUR_API_KEY', query: message }) }); const data = await response.json(); return data; }
